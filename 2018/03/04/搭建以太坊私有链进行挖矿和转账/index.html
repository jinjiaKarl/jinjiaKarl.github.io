建立以太坊私有链

  搭建私有网络最方便的方法就是通过geth命令中的–networkid选项,设置一个与主网不同的networkid(主网的networkid为1),这也是官方推荐的方法.
  系统和环境
    mac os 10.12.6
    geth（安装 brew tap ethereum/ethereum
     brew install ethereum）
    brew 是 Mac 下的包管理工具，和Ubuntu里的apt-get类似

  1. 首先创建一个目录
    mkdir private-geth
    cd private-geth

  2. 建立创世区块文件，是一个json格式的文件（因为如果使用默认值，difficulty会非常高，不好挖到矿），文件内容如下：

  3. 初始化创世纪节点,并设置data目录:
    geth –datadir ./data/00 init genesis.json
    4.启动节点, 加上console 表示启动后,启用命令行:
    geth –datadir ./data/00 –networkid 15 console 2>>geth.log
    参数解释：–datadir 后面的参数是区块数据及秘钥存放目录。
    console 进入控制台
    2>> test.log 表示把控制台日志输出到test.log文件
    为了更好的理解，建议新开一个命令行终端，实时显示日志：
    tail -f test.log

  4. 创建账户进行挖矿
    创建两个账户，括号里为账户的密码（牢记）

    开始挖矿，第一次会比较慢，打开test.log文件，看到如下信息：
    INFO [03-04|20:23:48] Generating DAG in progress epoch=0 percentage=0 elapsed=1.867s

INFO [03-04|20:23:50] Generating DAG in progress epoch=0 percentage=1 elapsed=3.670s

INFO [03-04|20:23:52] Generating DAG in progress epoch=0 percentage=2 elapsed=5.456s

INFO [03-04|20:23:53] Generating DAG in progress epoch=0 percentage=3 elapsed=7.218s

INFO [03-04|20:23:55] Generating DAG in progress epoch=0 percentage=4 elapsed=9.083s

  当percentage为100时，第一个区块完成，返回ture区块信息存储在./.ethash文件中，奖励给第一个账户5个以太币。

  我们拿第一个账户给第二个账户发送2个以太币（在发送以太币之前，我们到对第一个账户进行解锁），然后再进行挖矿，将交易写入区块，查看账户余额。
